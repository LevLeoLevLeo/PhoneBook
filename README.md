# PhoneBook

## Описание
Телефонный справочник, позволяющий пользователям создавать и изменять свои списки с телефонными номерами.

## Что реализуется
+ Дизайн интерфейса;
+ Диаграмма активности;
+ Диаграмма последовательности;
+ UseCase;
+ Проектирование БД;
+ Создание БД;
+ Создание приложения.
